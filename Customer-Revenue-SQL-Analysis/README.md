# Customer Revenue Analysis using SQL

## Project Overview
This project analyzes customer purchasing behavior and revenue trends using SQL.

## Tools Used
- SQL
- Data Analysis

## Dataset
The dataset contains order transactions including customer information, region, category, product, quantity, and revenue.

## Business Questions
- What is the total revenue generated?
- Which regions generate the highest sales?
- Who are the top customers?
- Which product categories perform best?
- What are the monthly revenue trends?
