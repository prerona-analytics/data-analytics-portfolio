
# Customer Churn Analysis (Python EDA)

## Project Overview
This project analyzes customer churn patterns using Python and exploratory data analysis.

## Tools Used
Python  
Pandas  
Seaborn  
Matplotlib  

## Key Insights
Customers with month-to-month contracts show higher churn.

Higher monthly charges correlate with higher churn risk.

Customers with longer tenure show stronger retention.
