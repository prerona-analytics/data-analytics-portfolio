# How to Upload This EdTech Analytics Project to GitHub

## Step-by-Step Guide (5 Minutes)

### 1. Create Repository on GitHub

- Go to https://github.com/new
- Repository name: `edtech-student-analytics`
- Description: "EdTech student learning analytics: engagement, performance, and retention analysis with advanced visualizations"
- Select **Public** (so recruiters can view)
- **DO NOT** initialize with README
- Click **Create Repository**

### 2. Upload Locally via Git

```bash
cd edtech_student_analytics_project

# Initialize git
git init
git add .
git commit -m "Initial commit: end-to-end edtech student analytics project"

# Set main branch
git branch -M main

# Add remote repository
git remote add origin https://github.com/YOUR_USERNAME/edtech-student-analytics.git

# Push to GitHub
git push -u origin main
```

Replace `YOUR_USERNAME` with your actual GitHub username.
