# Retail Sales Performance Dashboard

## Project Overview
This project analyzes retail sales performance using Power BI.  
The dashboard provides insights into sales trends, product performance, and regional sales distribution.

## Tools Used
- Power BI
- Excel ( CSV ) dataset
- Data Visualization (proof through screenshot/snip)

## Key Insights
- Sales performance across regions
- Category-wise sales comparison
- Profit analysis by category
- Top performing products
- Monthly sales trends
